Etiquetas semanticas

Las etiquetas semanticas ayudan a definir la estructura del documento y permiten que las paginas web sean mejor indexadas por los buscadores

Web responsive

Cuando utilizamos el termino responsive(adaptable), nos referimos principalmente a responsive design (diseño web adaptable).Esto significa hacer que un sitio web sea accesible y adaptable en todos los devices: smartphones, tablets, etc.
